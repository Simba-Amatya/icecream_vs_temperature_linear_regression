Ice Cream Sales Prediction

This project predicts daily ice cream sales based on temperature using Machine Learning.

The goal is to demonstrate:

Linear Regression for predicting sales

Experimentation with different models (Decision Tree)

Version control using Git for collaborative ML workflows

Dataset

The dataset contains daily temperature (°C) and ice cream sales ($) for 15 days.

Temperature: independent variable

Sales: dependent variable

*Note:* The dataset is small for demonstration purposes.
